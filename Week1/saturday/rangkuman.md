# Rangkuman Pemrograman Dasar

## 1. Apa itu javascript dan sejarahnya 
* **Apa itu javasript:** javascript adalah bahasa pemrograman yang melengkapi HTML dan CSS untuk membangun pengalaman web yang interaktif dan fungsional. Dengan memahami di mana JavaScript dijalankan dan standar penulisannya, kamu memiliki fondasi yang tepat untuk memulai perjalanan belajar JavaScript secara terstruktur.

* **Sejarahnya:** Pada awalnya, JavaScript hanya dirancang untuk dijalankan di dalam browser (Chrome, Firefox, Safari). Browser memiliki mesin khusus yang disebut JavaScript Engine — misalnya V8 di Chrome — yang bertugas membaca dan mengeksekusi kode JavaScript menjadi tindakan nyata di layar pengguna, Pengembangan yang berjalan di browser ini disebut sebagai pengembangan Front-End. Namun Pada tahun 2009, teknologi bernama Node.js diciptakan. Node.js adalah lingkungan (runtime environment) yang memungkinkan JavaScript dijalankan langsung di sistem operasi komputer atau server, tanpa memerlukan browser.

## 2. Variabel
**Variabel** adalah wadah di dalam memori komputer untuk menampung nilai atau data.
* **Fungsi:** Menyimpan data yang nilainya dapat diubah selama program berjalan.
* **Analogi:** Seperti kotak penyimpanan yang diberi label nama.
* **Peraturan dalam penulisan variabel:**
* -1.harus dalam bentuk satu kata.
* -2.tidak boleh pake spasi.
* -3.tidak bisa menggunakan simbol kecuali ( _ ) sama ( $ ).
* -4.tidak boleh didahului angka kecuali angka tersebut berada di tengah kata atau akhir kata.
* -dianjurkan menulis variabel dalam bentuk camelCast, dan penamaan yang masuk akal.

## 3. Tipe Data
**Tipe Data** adalah klasifikasi yang memberitahu komputer cara mengolah suatu nilai.
* **Fungsi:** Memastikan data diproses dengan benar sesuai jenisnya.
* **Contoh:** Membedakan angka untuk berhitung dan teks untuk nama.

## 4. Tipe Data String
**String** adalah tipe data kumpulan karakter yang membentuk teks (huruf, angka, spasi, simbol).
* **Syntax:** Ditulis wajib diapit tanda kutip tunggal (`'...'`) atau ganda (`"..."`).
* **Sifat:** Angka di dalam string dianggap teks biasa dan tidak bisa dihitung matematis.
* **Contoh:** `"Budi"`, `'081234'`, `"Kode_123!"`.

## 5. Syntax
**Syntax** adalah aturan penulisan dan tata bahasa baku dalam bahasa pemrograman.
* **Sifat:** Bersifat mutlak.
* **Dampak:** Kesalahan penulisan (salah syntax) menyebabkan program macet atau *error*.

## 6. Statement
**Statement** adalah satu instruksi utuh yang memerintahkan komputer melakukan tindakan.
* **Komponen:** Bisa berupa deklarasi variabel, operasi matematika, atau fungsi cetak.
* **Ciri:** Sering kali diakhiri tanda titik koma (`;`) pada bahasa pemrograman tertentu.
